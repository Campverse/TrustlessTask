# TrustlessTask User Guide

## Getting Started

### 1. Connect Your Wallet

TrustlessTask supports:
- Nami Wallet
- Eternl Wallet
- Flint Wallet

Click "Connect Wallet" and select your preferred wallet.

### 2. For Clients: Creating a Project

1. Click "Create Project"
2. Fill in project details:
   - Title and description
   - Freelancer's wallet address
   - Optional arbiter address
3. Add milestones:
   - Description of deliverable
   - Payment amount (in ADA)
   - Deadline
4. Review total amount
5. Sign transaction to lock funds in escrow

### 3. For Freelancers: Completing Work

1. View assigned projects
2. Work on milestone deliverables
3. Click "Mark Complete" when done
4. Wait for client approval
5. Receive payment automatically

### 4. For Clients: Approving Work

1. Review completed milestone
2. Verify deliverables meet requirements
3. Click "Approve & Release Funds"
4. Sign transaction to release payment

## Dispute Resolution

If there's a disagreement:

1. Either party clicks "Raise Dispute"
2. Provide detailed reason
3. Arbiter reviews evidence
4. Arbiter makes binding decision
5. Funds distributed per outcome

## Reputation System

Your on-chain reputation includes:
- Completed projects
- Total earned/spent
- Average rating
- Dispute history

Build reputation by:
- Completing projects successfully
- Meeting deadlines
- Resolving issues professionally
- Avoiding disputes

## Best Practices

### For Clients
- Write clear project descriptions
- Set realistic milestones
- Respond promptly to completions
- Provide constructive feedback

### For Freelancers
- Communicate progress regularly
- Meet agreed deadlines
- Deliver quality work
- Document your work

## Safety Tips

- Verify wallet addresses carefully
- Start with smaller projects
- Check reputation scores
- Use arbiter for large projects
- Keep communication records
